Uma coleção curada de skills especializadas projetadas para turbinar assistentes modernos de programação como **Antigravity** e **Cursor**. Este repositório fornece fluxos de trabalho avançados e um servidor MCP global.

## 📂 Skills Disponíveis

- **[📱 Android Expert](./android-expert)**: Melhores práticas para desenvolvimento Android (Java/Kotlin, Material Design).
- **[✅ Code Review Standards](./code-review-standards)**: Revisões de PEP 8, types, Docker e testes.
- **[🎨 Dashboard Cloner](./dashboard-cloner)**: Análise de UI/UX para extrair design tokens e layouts.
- **[🐍 Django Audit](./django-audit)**: Auditoria técnica profunda para aplicações Django.
- **[🐙 Git Platforms Expert](./git-platforms-expert)**: Fluxos de trabalho GitHub/GitLab e CI/CD.
- **[🏗️ New App Bootstrap](./new-app-bootstrap)**: Scaffolding inicial para novas aplicações.
- **[📋 Phase Execution](./phase-execution)**: Metodologia de entrega estruturada em 5 fases.
- **[🌍 README Bilíngue](./readme-bilingual)**: Documentação dual-language automatizada.
- **[📝 Technical Documentation](./technical-documentation)**: Criação padronizada de `analise.md` e `plano.md`.
- **[🔄 Update Funcoes.md](./update-funcoes-md)**: Rastreamento automático de funções em `funcoes.md`.

## 🚀 Como Começar

Você pode usar estas skills de duas formas:

### 1. Global (MCP) - Recomendado
Rodar o servidor MCP incluso garante acesso instantâneo e global a todas as skills.
- **Setup no Antigravity**: [English](./mcp-server/antigravity_setup.md) | [Português](./mcp-server/antigravity_setup.pt-BR.md)
- **Setup no Cursor**: [English](./mcp-server/cursor_setup.md) | [Português](./mcp-server/cursor_setup.pt-BR.md)

### 2. Local (Cópia Manual)
Se preferir adicionar skills individualmente por projeto:
- **Pastas**: Copie a pasta da skill para `.agents/skills/` (para o Antigravity).
- **Regras**: Copie o conteúdo para `.cursorrules` (para o Cursor).

Para detalhes sobre a implementação do servidor, veja o [README do mcp-server](./mcp-server/README.md).

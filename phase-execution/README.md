Guides project delivery in five phases (analysis, planning, implementation, validation, delivery) with checklists and a structured final report. The agent runs only the requested phase and produces status updates and a delivery report.

## When to use

Use when the user asks to **execute a phase**, **deliver Phase 1**, run a **methodological delivery**, or follow **phased execution**.

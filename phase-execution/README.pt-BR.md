![Cursor Skill](https://img.shields.io/badge/Cursor-Skill-00d4aa?style=flat) ![Markdown](https://img.shields.io/badge/doc-Markdown-083fa1?style=flat) ![regras-ia](https://img.shields.io/badge/regras--ia-rules-6366f1?style=flat)

Leia em inglês: [README.md](README.md)

Orienta a entrega do projeto em cinco fases (análise, planejamento, implementação, validação, entrega) com checklists e relatório final estruturado. O agente executa apenas a fase solicitada e produz atualizações de status e relatório de entrega.

## Quando usar

Use quando o usuário pedir **executar uma fase**, **entregar a Fase 1**, rodar **entrega metodológica** ou seguir **execução por fases**.
